## 🇪🇸 Versión en Español

### 🚀 Descripción General (No técnica)
Esta herramienta permite cargar imágenes y aplicar transformaciones matemáticas para mejorar su contraste y visualizar su composición. Es ideal para entender cómo las computadoras procesan imágenes tratándolas como matrices numéricas.
*   **¿Cómo usarla?** Simplemente haz clic en el enlace de arriba, sube una imagen y selecciona el filtro que desees aplicar.
*   **Funciones:** Mejora de contraste (Expansión lineal), resaltado de detalles (Ecualización) y gráficos de distribución de luz.

### 💻 Instalación y Ejecución (Técnico)
Si eres desarrollador y deseas ejecutar el código localmente usando **Python 3.10**, sigue estos pasos:

**1. Crear un entorno virtual (venv)**
Esto garantiza que las librerías no entren en conflicto con otros proyectos.
*   **Windows:**
    ```bash
    py -3.10 -m venv venv
    .\venv\Scripts\activate
    ```
*   **macOS / Linux:**
    ```bash
    python3.10 -m venv venv
    source venv/bin/activate
    ```

**2. Instalar dependencias**
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

**3. Ejecutar la aplicación**
```bash
streamlit run streamlit_app.py
```

## 🌎 English Version

### 🚀 Overview (Non-Technical)
This tool allows you to upload images and apply mathematical transformations to improve contrast and visualize their composition. It demonstrates how computers process images by treating them as numerical matrices.
*   **How to use it?** Just click the link at the top, upload an image, and choose the filter you want to apply. No installation is required for the web version.
*   **Features:** Contrast enhancement (Linear Expansion), detail highlighting (Equalization), and light distribution charts.


